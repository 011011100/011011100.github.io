## 介绍

这就是一个基于Vitepress的主题，它是一个简单的博客主题，可以使用MarkDown文件快速的在页面上渲染，成为属于个人的博客项目。

## 下载项目

```bash
# 下载项目
git clone git@github.com:elonehoo/vitepress-theme-simple.git

# 进入目录
cd vitepress-theme-simple

# 安装依赖
pnpm install
```
